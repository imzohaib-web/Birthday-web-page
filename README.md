# Birthday Web Page

A modern and colorful birthday webpage made with HTML and CSS. It includes a birthday greeting, profile photo, soft pastel styling, smooth animations, and interactive gift cards that reveal images when hovered.

## Features

- Clean pastel color palette
- Responsive layout for desktop and mobile screens
- Smooth profile image animation
- Hover effects on gift cards
- Simple HTML and CSS structure
- Easy to customize with your own images and text

## Project Files

- `index.html` - Main webpage structure
- `style.css` - Page design, layout, colors, and animations
- `dp.jpg` - Profile image
- `images.jpg` - Default gift cover image
- `secret.jpg`, `react.jpg`, `genius.jpg`, `giphy.webp` - Images shown when hovering over gifts

## How To Use

1. Open the project folder.
2. Open `index.html` in your browser.
3. Hover over the gift boxes to reveal the birthday surprises.

## Customize

You can edit the text in `index.html` to change the name, age, date, and gift messages.

To change the images, replace the image files in the folder or update the image names in `style.css`.
